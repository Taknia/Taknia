Installation Instructions

  1. Download the Module
    • Download the Streamlined User Account Menu module from the Odoo App Store.
    • Save the downloaded module file (usually a .zip file) to your Odoo addons.
    • Log in to your Odoo system as an Administrator.
    • Navigate to the Apps module on your Odoo dashboard.
  2. Install the Module
    • Once the module is uploaded, it will appear in the Apps list.
    • Click the Install button next to the Streamlined User Account Menu module.
    • Odoo will install the module for you. Once it's done, the module is ready to use!
