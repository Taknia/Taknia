✅ ملف الشرح (Documentation)
📌 وصف الموديول
موديول تقارير محاسبية متقدمة يدعم:

التقارير المالية الرئيسية (ميزان المراجعة - كشف الحسابات - القيود).
دعم فروع متعددة (Branches).
دعم فلترة بالشركات - الحسابات - العملات - الشركاء - اليوميات.
تقارير PDF وExcel.
دعم اللوجو واسم الشركة.
📌 التقارير المتاحة
التقرير المحاسبي العام (Accounting Report)

ميزان مراجعة.
كشف حسابات تفصيلي.
القيود اليومية (Journal Entries).
جميع الحسابات أو محددة.
لكل الفروع أو فرع معين.
لكل الشركاء أو شريك معين.
لكل العملات أو عملة معينة.
لكل اليوميات أو يومية معينة.
كشف حساب الشركاء (Partner Ledger Report)

كشف مفصل لكل شريك.
لكل الفروع أو فرع معين.
لكل العملات أو عملة معينة.
لكل الشركاء أو شريك معين.
تصدير PDF وExcel.
📌 الفلاتر (Filters)
كل التقارير تدعم:

تحديد فرع معين أو جميع الفروع.
تحديد شريك معين أو جميع الشركاء.
تحديد عملة معينة أو جميع العملات.
تحديد يومية معينة أو جميع اليوميات.
تحديد الحسابات من - إلى.
تحديد التاريخ من - إلى.
