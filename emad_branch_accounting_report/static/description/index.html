Branch Accounting Report Module - EMAD Kadry Taknia Soft Description: --------------- This custom module provides advanced branch accounting reports for Odoo 18. It supports: - Multi-branch structure (Company with Branches) - Multiple filters: Branch, Partner, Journals, Accounts, Currency - Output in PDF and Excel formats Installation: --------------- 1. Copy folder "branch_accounting_report" into your Odoo custom addons directory. 2. Restart Odoo service. 3. Activate developer mode. 4. Go to Apps > Update App List > Search "Branch Accounting Report - EMAD Kadry" > Install. Usage: --------------- 1. Go to "Accounting" menu. 2. Under "Reporting", you will see "Branch Accounting Report". 3. Open the wizard, fill filters, and choose either "Print PDF" or "Export Excel". Filters: --------------- - Date From / Date To - Branches (Multiple selection) - Partners (Multiple selection) - Journals (Multiple selection) - Accounts (Multiple selection) - Currency (Single selection) Report Output: --------------- - PDF report with filters applied. - Excel report (optional - can be extended in future).
